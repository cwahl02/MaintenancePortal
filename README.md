## Overview

MaintenancePortal provides:
- ✅ **Ticket Management** - Create, track, and resolve maintenance requests
- ✅ **User System** - Registration, authentication, and profile management
- ✅ **Status Tracking** - Monitor tickets through their lifecycle (Open, In Progress, Completed)
- 🚧 **Advanced Features** - Tags, comments, permissions, and more coming soon!

### Upcoming Features:
- 🚧 Ticket tags (GitHub-style labels)
- 🚧 Comment system
- ⏳ Group-based tickets
- ⏳ User dashboard
- ⏳ Jobs and permission system
- ⏳ Audit logs
- ⏳ Theme system
- ⏳ Notifications and reporting

### Feature Status Legend:
- ✅ **Complete** - Feature is fully implemented and available
- 🚧 **In Progress** - Feature is currently being developed
- ⏳ **Planned** - Feature is planned for future development

See the full [Features & Roadmap](FEATURES.md) for details.

## Technologies

### Core Stack:
- **.NET 9 SDK** - Core framework
